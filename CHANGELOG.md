# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.3-beta] - 2024-12-04

### 新增
- 多课表管理功能（创建、切换、重命名、删除）
- 周视图滑动切换
- 自定义主题颜色（背景色、字体色、课程方框色）
- 作息时间自定义设置
- 连堂课支持（可设置开始节次和结束节次）
- 设置界面（常规设置、颜色设置、作息时间设置、关于）
- 页面切换动画效果
- 侧滑返回手势支持

### 优化
- 设置界面固定白底黑字，提升可读性
- 下拉选择器替代文本输入，操作更便捷
- 课程方块连续显示，视觉效果更佳

## [0.2-alpha] - 2024-12-03

### 新增
- 基础课表显示功能
- 课程添加、编辑、删除
- 课程详情查看
- 数据本地持久化存储

## [0.1-alpha] - 2024-12-02

### 新增
- 项目初始化
- 基本 UI 框架搭建
